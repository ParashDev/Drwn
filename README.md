# drwn

A browser-based design tool for creating social media graphics, posters, presentations, and more. No server required — runs entirely in the browser.

## Features

- **Shapes & Elements** — Rectangles, circles, triangles, stars, lines, and 20+ extra shapes
- **Text** — Rich text with font families, weights, sizes, colors, alignment, letter spacing, and line height
- **Images** — Upload images, clip images inside shapes, apply filters (brightness, contrast, saturation, blur, etc.)
- **Templates** — 50+ built-in templates for posts, stories, reels, presentations, and more
- **Layers** — Drag-to-reorder layer panel, bring to front/back, lock/unlock, show/hide
- **Multi-select** — Marquee selection, group/ungroup, multi-drag with snap guides
- **Canvas** — Custom sizes, background color, zoom, pan, center guides, grid overlay
- **Export** — PNG, JPG, SVG, and PDF export
- **Save/Load** — Save designs to localStorage, auto-save on every change
- **Undo/Redo** — Full undo/redo history
- **Mobile** — Touch-optimized with pinch-to-zoom, bottom sheet panels, and large touch targets

## Getting Started

Just open `index.html` in a browser. No build step, no dependencies, no server.

## Tech

Pure HTML, CSS, and vanilla JavaScript. Zero dependencies.
